I'm an **Applied AI & Software Engineer**. I build AI search and recommendation systems that hold up in production.

- **Semantic search** that ranks by meaning, not keywords.
- **AI personalization** that distills user activity into recommendation signals.
- **Event-driven pipelines** that re-embed only what actually changed.
- **LLM workflows** that keep the model firmly in its lane.

The model call is one line. Building around it so it survives real users and real data is the actual job.
